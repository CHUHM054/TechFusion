1. 设：\(x(n) = \{5,4,3,2,1,5,4,3,2,1,\cdots,5,4,3,2,1\}_{[0,59]}\)
\(h(n) = \{1,3,2\}_{[0,2]}\) 求：\(y(n) = h(n) * x(n)\)
(分别用重叠相加法和重叠保留法)

**Step1：确定参数和分段长序列**

\(L = 60\) （输入序列 \(x(n)\) 的总长度）
\(M = 3\) （冲激响应 \(h(n)\) 的点数）
\(N = 5\) （根据 \(x(n)\) 的循环周期 5 所设定的输入信号分段长度）
\(x_i(n) = \{5, 4, 3, 2, 1\}, \quad i = 0, 1, \dots, 11\) （将长度为 60 的输入序列 \(x(n)\) 分为 12 段，每段即为一个循环周期的 \(x_i(n)\)）

**Step2：求每一段的卷积（重叠相加法）**

由于 \(x(n)\) 严格按周期 5 循环，所有 12 个输入分段 \(x_i(n)\) 均与第 0 段一致：
\[
x_i(n) = \{5, 4, 3, 2, 1\}, \quad i = 0, 1, \dots, 11
\]

采用两种方法交叉验证，求每一段的卷积 \(y_i(n) = x_i(n) * h(n)\)：

**方法一：时域直接线性卷积**
根据线性卷积定义 \(y_i(n) = \sum_{k=0}^{4} x_i(k)h(n-k)\)，逐项计算：
* \(n=0: \quad 5 \times 1 = 5\)
* \(n=1: \quad 5 \times 3 + 4 \times 1 = 19\)
* \(n=2: \quad 5 \times 2 + 4 \times 3 + 3 \times 1 = 25\)
* \(n=3: \quad 4 \times 2 + 3 \times 3 + 2 \times 1 = 19\)
* \(n=4: \quad 3 \times 2 + 2 \times 3 + 1 \times 1 = 13\)
* \(n=5: \quad 2 \times 2 + 1 \times 3 = 7\)
* \(n=6: \quad 1 \times 2 = 2\)

得到卷积结果：
\[
y_i(n) = \{5, 19, 25, 19, 13, 7, 2\}
\]

**方法二：补零至 \(P\) 点利用 DFT/FFT（交叉验证）**
为避免循环卷积混叠，需补零至点数 \(P = N + M - 1 = 5 + 3 - 1 = 7\)。
补零后得到：
\[
\tilde{x}_i(n) = \{5, 4, 3, 2, 1, 0, 0\}
\]
\[
\tilde{h}(n) = \{1, 3, 2, 0, 0, 0, 0\}
\]

进行 7 点 DFT：
\[
\tilde{X}_i(k) = \sum_{n=0}^{6} \tilde{x}_i(n) W_7^{kn}, \quad \tilde{H}(k) = \sum_{n=0}^{6} \tilde{h}(n) W_7^{kn}
\]

频域相乘：
\[
\tilde{Y}_i(k) = \tilde{X}_i(k) \tilde{H}(k)
\]

进行 7 点 IDFT 得到时域结果：
\[
\tilde{y}_i(n) = \frac{1}{7} \sum_{k=0}^{6} \tilde{Y}_i(k) W_7^{-kn}
\]

此时 7 点循环卷积结果将严格等于线性卷积结果，即：
\[
\tilde{y}_i(n) = y_i(n) = \{5, 19, 25, 19, 13, 7, 2\}
\]

**Step3：重叠相加法叠加各段结果**

**第一段叠加 (\(i=0\))：**
起始位置 \(n=0\)，无前段重叠，直接输出整段线性卷积结果：
\[
y(n) = \{5, 19, 25, 19, 13, 7, 2\}, \quad n \in [0, 6]
\]

**中间段叠加 (\(i=1 \sim 10\))：**
取普适的第 \(i\) 段。由于 \(y_i(n)\) 的前两项 \([5, 19]\) 与上一段 \(y_{i-1}(n)\) 的后两项 \([7, 2]\) 在位置 \(5i\) 处发生重叠，叠加后形成周期性的结果：
\[
y(5i) = 5 + 7 = 12
\]
\[
y(5i+1) = 19 + 2 = 21
\]
后续无重叠，直接承接普适段的中间项及尾项：
\[
y(5i+2) = 25,\quad y(5i+3) = 19,\quad y(5i+4) = 13,\quad y(5i+5) = 7,\quad y(5i+6) = 2
\]
*(此规律适用于中间所有段)*

**最后一段叠加 (\(i=11\))：**
输入位置从 \(n=55\) 开始，先与前段（\(i=10\)）进行重叠相加：
\[
y(55) = y_{11}(0) + y_{10}(5) = 5 + 7 = 12
\]
\[
y(56) = y_{11}(1) + y_{10}(6) = 19 + 2 = 21
\]
后续无后续段的叠加，直接取出 \(y_{11}(n)\) 的剩余项作为最终结果：
\[
y(57) = 25,\quad y(58) = 19,\quad y(59) = 13,\quad y(60) = 7,\quad y(61) = 2
\]

**最终输出结果 \(y(n)\)：**
综合上述三段，\(y(n) = h(n) * x(n)\) 在 \(n=0 \sim 61\) 的输出为：
\[
y(n) = \{5, 19, 25, 19, 13, \underbrace{12, 21, 25, 19, 13, 7, 2, \cdots, 12, 21, 25, 19, 13, 7, 2}_{i=1 \text{ 到 } 11 \text{ 的迭代}}, 12, 21, 25, 19, 13, 7, 2\}
\]

**Step1：确定参数和分段长序列**

\(L = 60\) （输入序列 \(x(n)\) 的总长度）
\(M = 3\) （冲激响应 \(h(n)\) 的点数）
\(N = 5\) （响应信号分段长度，即重叠保留法中截取的有效输出点数，分段标准与重叠相加法一致）
\(P = N + M - 1 = 7\) （由于对输出 \(y\) 分段，重叠保留法中实际对输入 \(x\) 的分段及DFT点数）

\(x_i(n)\) 为重叠保留法的第 \(i\) 段输入序列，共需 \(i = 0, 1, \dots, 12\) 段（共13段），每段长度为 \(P=7\)。其定义如下：
\[
x_0(n) = \{0, 0, 5, 4, 3, 2, 1\}
\]
\[
x_i(n) = \{x(5i-2), x(5i-1), x(5i), x(5i+1), x(5i+2), x(5i+3), x(5i+4)\}, \quad 1 \le i \le 11
\]

特别说明：由于需计算 \(y(n)\) 尾部 \(60 \le n \le 64\) 的结果，因此存在 \(i=12\) 的情况。\(x_{12}(n)\) 取值如下（包含 \(x(n)\) 尾部的非零值，序列不全为零）：
\[
x_{12}(n) = \{x(58), x(59), 0, 0, 0, 0, 0\} = \{2, 1, 0, 0, 0, 0, 0\}
\]

**Step2：求每一段的卷积（重叠保留法）**

分段情况分为三类：**第0段（\(i=0\)）**、**中间段（\(i=1 \sim 11\)）**、**最后一段（\(i=12\)）**。采用两种方法交叉验证。

**一、第0段（\(i=0\)）**
输入序列 \(x_0(n) = \{0, 0, 5, 4, 3, 2, 1\}\)

**方法一：时域直接线性卷积（交叉验证）**
\[
y_{lin,0}(n) = x_0(n) * h(n) = \{0, 0, 5, 19, 25, 19, 13, 7, 2\}
\]

**方法二：7点循环卷积（DFT定义，不补零）**
\[
y_{circ,0}(n) = \{7, 2, 5, 19, 25, 19, 13\}
\]
（其结果为线性卷积后 \(n=0,1\) 项混叠上前两项 \(y_{lin,0}(7), y_{lin,0}(8)\) 之和）
**有效输出（保留后 \(N=5\) 点，舍弃前 \(M-1=2\) 点）：**
\[
y_{eff,0}(n) = \{5, 19, 25, 19, 13\}
\]

---

**二、中间段（\(i=1 \sim 11\)）**
输入序列均为 \(x_i(n) = \{2, 1, 5, 4, 3, 2, 1\}\)

**方法一：时域直接线性卷积（交叉验证）**
\[
y_{lin,i}(n) = x_i(n) * h(n) = \{2, 7, 12, 21, 25, 19, 13, 7, 2\}
\]

**方法二：7点循环卷积（DFT定义，不补零）**
\[
y_{circ,i}(n) = \{9, 9, 12, 21, 25, 19, 13\}
\]
**有效输出（保留后 \(N=5\) 点，舍弃前 \(M-1=2\) 点）：**
\[
y_{eff,i}(n) = \{12, 21, 25, 19, 13\}
\]

---

**三、最后一段（\(i=12\)）**
输入序列 \(x_{12}(n) = \{2, 1, 0, 0, 0, 0, 0\}\) （提取尾部 \(x(58), x(59)\)）

**方法一：时域直接线性卷积（交叉验证）**
\[
y_{lin,12}(n) = x_{12}(n) * h(n) = \{2, 7, 7, 2, 0, 0, 0, 0, 0\}
\]

**方法二：7点循环卷积（DFT定义，不补零）**
\[
y_{circ,12}(n) = \{2, 7, 7, 2, 0, 0, 0\}
\]
**有效输出（保留后 \(N=5\) 点，舍弃前 \(M-1=2\) 点）：**
\[
y_{eff,12}(n) = \{7, 2, 0, 0, 0\}
\]

**Step3：拼接各段有效输出（重叠保留法的最终组合）**

重叠保留法的输出不需要进行“相加”，而是将各段计算后保留的有效部分（舍弃前 \(M-1\) 点），根据索引位置直接拼接组合。

**第一段 (\(i=0\))：**
\(y_{eff,0}(n) = \{5, 19, 25, 19, 13\}\)，对应输出索引 \(n \in [0, 4]\)。

**中间段 (\(i=1 \sim 11\))：**
取普适的第 \(i\) 段。
\(y_{eff,i}(n) = \{12, 21, 25, 19, 13\}\)，对应输出索引 \(n \in [5i, 5i+4]\)。
该序列从 \(i=1\) 到 \(i=11\) 共重复 11 次，对应输出索引 \(n \in [5, 59]\)。

**最后一段 (\(i=12\))：**
\(y_{eff,12}(n) = \{7, 2, 0, 0, 0\}\)，对应输出索引 \(n \in [60, 64]\)。

---

**最终输出结果 \(y(n)\)：**
将上述三段按顺序拼接，得到重叠保留法的最终结果 \(y(n) = h(n) * x(n)\)，其长度为 \(L+M-1=62\)（实际序列截至索引 61 变为 0）：

\[
y(n) = \{ \underbrace{5, 19, 25, 19, 13}_{n=0\sim4}, \underbrace{12, 21, 25, 19, 13, \cdots, 12, 21, 25, 19, 13}_{n=5\sim59\text{，共 11 组重复}}, \underbrace{7, 2, 0, 0, 0}_{n=60\sim64} \}
\]
