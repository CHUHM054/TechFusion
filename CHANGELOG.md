# 系统更新日志

## Ver 0.9.2 (Build 6) - 2026-06-18
### 🐛 修复
- 页面切换延迟修复：`fetch_online_quotes` 改为异步后台线程+30分钟节流，消灭5s阻塞
- 填空多值答案识别：支持逗号/空格分隔多空输入，逐项匹配标准答案

### ✨ 体验优化
- UI布局重平衡：计分栏压缩为紧凑单行caption，去掉分隔线，题目区获得更多空间
- 字体舒适度提升：移除SimSun锯齿字体，题目24px+行距1.6em，全局行距1.8

### 🔧 调优
- `load_questions` 缓存TTL 5min→30min，减少CSV重读
- `FETCH_TIMEOUT` 5s→2s，`FETCH_COUNT` 5→2

## Ver 0.9.1 (Build 5) - 2026-06-18
### 🐛 修复
- 抽样随机性修复：种子改用毫秒时间戳，消除连续答题题目重复
- 错题本精确过滤：结果页只显示本轮答错的题，不混入历史错题
- 仪表盘模式名 markdown `**` 渲染异常 → 改用 `###` 标题

### ✨ 优化
- KaTeX 公式渲染支持：题目/答案/解析区支持 `$...$` LaTeX 渲染
- 填空题答案人性化：CSV answer 列允许 `$U_{95}$` 等价表述，normalize 自动剥离 LaTeX 标记
- 页面切换性能优化：`load_questions()` 加 `@st.cache_data(ttl=300)`，4页面预加载热缓存
- 版本号/题库更新日期刷新至 2026-06-18
