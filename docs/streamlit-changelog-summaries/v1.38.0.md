# Streamlit 1.38.0 更新摘要

**发布日期:** 2024-08-27

## Highlights
- `st.dataframe` / `st.data_editor` 扩展支持 Polars、xarray、Dask、Ray、DB API 2.0、Pydantic
- 新增 dataframe interchange protocol 支持
- `st.json` 支持可配置展开深度

## Notable Changes
- 移除 experimental cached widget replay 逻辑
- `st.code` 支持自动换行
- Plotly 工具栏按钮可自定义

## 对当前项目的潜在影响 / 应用建议
- 若后续使用 Polars 处理排行榜数据，`st.dataframe` 可直接渲染
- `st.json` 展开深度可用于调试展示题目元数据
