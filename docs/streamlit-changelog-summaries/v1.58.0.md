# Streamlit 1.58.0 更新摘要

**发布日期:** 2026-05-28

## Highlights
- 新增 `st.pagination` 分页组件
- `@st.fragment` 支持 `parallel=True` 并行调度
- `st.expander` 与 `st.status` 新增 `type` 参数（紧凑样式）

## Notable Changes
- 移除已弃用的 `add_rows` 功能
- 移除 langchain callback handler 集成
- URL 同步支持 query-param 绑定 widget 的 session_state 更新

## 对当前项目的潜在影响 / 应用建议
- `st.pagination` 可直接替换错题本/排行榜的手动分页逻辑
- `@st.fragment(parallel=True)` 可用于并行刷新排行榜与 localStorage 后台任务
- `st.status(type='compact')` 适合轻量化展示题库加载进度
