# Streamlit 1.32.0 更新摘要

**发布日期:** 2024-03-07

## Highlights
- 新增 `st.popover` 布局容器，支持弹出式内容
- `st.radio` 选项支持 Markdown 链接
- `st.write` 支持直接传入 PIL 图像

## Notable Changes
- 懒加载 pandas、pyarrow、numpy，提升启动性能
- 修复 `st.page_link` / `st.switch_page` 路径处理
- 修复自定义组件主题样式注入

## 对当前项目的潜在影响 / 应用建议
- `st.popover` 可用于错题本/排行榜的筛选器折叠，节省页面空间
- 懒加载优化对本地启动速度有帮助
