# Streamlit 1.43.0 更新摘要

**发布日期:** 2025-03-04

## Highlights
- `st.chat_input` 集成文件上传
- 主题新增 `roundness`、`borderColor`、`linkColor`、`baseFontSize`、自定义字体等配置
- `st.dataframe` / `st.data_editor` 默认启用 `use_container_width=True`

## Notable Changes
- `rich` 从必需依赖中移除
- 支持 Polars 数据框/序列哈希
- 新增 JSON 列类型

## 对当前项目的潜在影响 / 应用建议
- 主题圆角/边框配置可统一答题系统视觉风格
- 若接入 AI 答疑，`st.chat_input` 文件上传可直接复用
