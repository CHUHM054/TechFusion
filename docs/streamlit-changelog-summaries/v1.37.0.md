# Streamlit 1.37.0 更新摘要

**发布日期:** 2024-07-25

## Highlights
- `st.fragment` 与 `st.dialog` 去实验化（de-experimentalize）
- 新增 `st.feedback` 评分组件
- Markdown 支持 Material 图标

## Notable Changes
- 支持 numpy 2.x
- 移除大量已弃用的 experimental 功能
- `st.bar_chart` / `st.area_chart` 支持堆叠选项

## 对当前项目的潜在影响 / 应用建议
- 可直接使用稳定的 `st.dialog` 做弹窗，`st.fragment` 做局部刷新
- `st.feedback` 可用于答题后对题目难度打分
