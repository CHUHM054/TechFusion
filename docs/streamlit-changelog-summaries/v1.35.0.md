# Streamlit 1.35.0 更新摘要

**发布日期:** 2024-05-23

## Highlights
- 新增 `st.logo`，可在侧边栏展示应用 Logo
- `st.dataframe` 支持行/列选择
- `st.plotly_chart`、`st.altair_chart`、`st.vega_lite_chart` 支持选择交互

## Notable Changes
- 修复自定义组件独立执行问题
- 页面切换时重置 widget 状态
- 允许 protobuf v5（后续版本撤销）

## 对当前项目的潜在影响 / 应用建议
- 可用 `st.logo` 为答题系统添加品牌标识
- `st.dataframe` 选择能力可为排行榜/错题数据提供交互
