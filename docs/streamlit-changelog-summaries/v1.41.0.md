# Streamlit 1.41.0 更新摘要

**发布日期:** 2024-12-10

## Highlights
- `st.write` 支持 pathlib 路径对象
- `st.write_stream` 支持异步生成器
- `st.text_input` / `st.date_input` / `st.time_input` 支持 ISO 时间/日期格式

## Notable Changes
- 支持 Python 3.13
- 异常堆栈不再暴露内部 Streamlit 细节
-  dataframe 支持通过列配置固定列

## 对当前项目的潜在影响 / 应用建议
- 若题目资源使用本地路径，`st.write(Path(...))` 可直接显示
- 异常信息更干净，便于用户排查题库加载问题
