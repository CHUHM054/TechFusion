# Streamlit 1.45.0 更新摘要

**发布日期:** 2025-04-29

## Highlights
- `st.context` 新增 `url`、`ip_address`、`is_embedded`
- `st.text_input` / `st.number_input` 支持 `icon` 参数
- `st.selectbox` / `st.multiselect` 支持动态新增选项 `accept_new_options`

## Notable Changes
- Markdown 锚点改为 slugify 格式
- 警告/异常元素支持 `width` 参数
- 优化单字符串 `st.write` 性能

## 对当前项目的潜在影响 / 应用建议
- `st.context` 可获取用户 IP/URL，用于排行榜去重或审计
- 输入框图标可增强搜索/筛选界面的可用性
