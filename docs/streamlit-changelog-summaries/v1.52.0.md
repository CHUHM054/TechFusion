# Streamlit 1.52.0 更新摘要

**发布日期:** 2025-12-04

## Highlights
- `st.container` 支持 `width='content'`
- `st.dataframe` / `st.data_editor` 支持 `height='content'`
- `st.file_uploader` / `st.camera_input` 改用 `key` 作为主要标识

## Notable Changes
- 移除原生 `st.bokeh_chart` 支持
- `st.vega_lite_chart` 弃用 kwargs
- `st.chat_input` 支持音频录制 `accept_audio`

## 对当前项目的潜在影响 / 应用建议
- `height='content'` 让错题列表 dataframe 高度自适应内容
- 文件上传以 key 为标识后，多文件场景状态管理更稳定
