# Streamlit 1.34.0 更新摘要

**发布日期:** 2024-05-02

## Highlights
- 新增 `@st.experimental_dialog` 对话框装饰器
- 缓存函数支持按参数清除：`cached_func.clear(arg)`
- `st.audio` / `st.video` 支持自动播放

## Notable Changes
- Toast 通知移动到右上角并优化样式
- `st.status` 空状态修复
- 支持 Modin 与 Snowpark Pandas

## 对当前项目的潜在影响 / 应用建议
- `st.experimental_dialog` 可用于确认交卷、删除记录等交互弹窗
- Toast 样式改进后，`st.toast` 提示更美观
