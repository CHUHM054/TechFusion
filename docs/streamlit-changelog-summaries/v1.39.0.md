# Streamlit 1.39.0 更新摘要

**发布日期:** 2024-10-01

## Highlights
- `st.button`、`st.download_button`、`st.form_submit_button`、`st.link_button`、`st.popover` 均支持 `icon` 参数
- `st.logo` 支持尺寸选项
- `st.form` 新增 `enter_to_submit` 参数

## Notable Changes
- 可通过 `key` 为元素设置 CSS 类
- 新增会话断开 TTL 配置
- 支持 watchdog 5.x

## 对当前项目的潜在影响 / 应用建议
- 按钮加图标可提升答题系统 UI 辨识度（如“下一题”箭头图标）
- `enter_to_submit` 可让答题表单一键提交，提升作答效率
