# Streamlit 1.57.0 更新摘要

**发布日期:** 2026-04-28

## Highlights
- Starlette 成为默认服务器并移除 Tornado
- 新增 `st.bottom` 底部固定容器
- `st.App` 暴露到 `st` 命名空间，支持自定义错误处理与 secrets 参数

## Notable Changes
- `plotly_chart` / `vega_lite_chart` 移除已弃用 kwargs
- 移除 `_get_websocket_headers`
- Polars 直接转 Arrow，跳过 pandas

## 对当前项目的潜在影响 / 应用建议
- `st.bottom` 适合放置交卷/下一题等常驻操作按钮
- Starlette 默认化后，本地运行更轻量，但需确认部署环境兼容
