# Streamlit 1.47.0 更新摘要

**发布日期:** 2025-07-16

## Highlights
- `st.html`、`st.feedback`、`st.pills`、`st.segmented_control` 等新增 `width` 参数
- 主题新增 `codeFontSize`、`linkUnderline`、`baseFontWeight`、`dataframeHeaderBackgroundColor` 等选项
- `st.code` 支持 `width='content'`

## Notable Changes
- 垂直布局中 `stretch` / `fixed` 高度行为变更（Breaking）
- `st.metric` 支持 `height` 参数
- 隐藏未显式设置的 about 对话框

## 对当前项目的潜在影响 / 应用建议
- `st.pills` / `st.segmented_control` 宽度参数便于顶部筛选栏自适应
- 主题变量增多，可更精确控制代码块、链接样式
