# Streamlit 1.36.0 更新摘要

**发布日期:** 2024-06-20

## Highlights
- 新增 `st.navigation` 与 `st.Page`，官方多页应用 v2 导航
- `st.bar_chart` 支持水平展示
- `st.columns` 新增 `vertical_alignment` 参数

## Notable Changes
- 移除旧版缓存逻辑（legacy caching）
- 弃用 `experimental_allow_widgets` 缓存参数
- Streamlit Charts 支持自定义坐标轴标签

## 对当前项目的潜在影响 / 应用建议
- 本项目当前最低版本即为 1.36，`st.navigation` 是可选升级点
- 水平柱状图可用于智能分析页面展示统计结果
