# Streamlit 1.33.0 更新摘要

**发布日期:** 2024-04-04

## Highlights
- 新增 `@st.experimental_fragment` 装饰器，支持局部重渲染
- 新增 `st.html` 命令，用于渲染原始 HTML
- `st.query_params` 新增显式 `update()` 方法

## Notable Changes
- `st.write` 中 `unsafe_allow_html` 必须显式声明
- 媒体元素（audio/video）改进
- 支持 packaging 24.x

## 对当前项目的潜在影响 / 应用建议
- `@st.fragment` 为后续局部刷新（如答题计时、自动保存）打下基础
- `st.html` 可替代当前 `components.html` 做 localStorage 注入，但需评估兼容性
