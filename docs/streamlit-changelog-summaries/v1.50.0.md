# Streamlit 1.50.0 更新摘要

**发布日期:** 2025-09-23

## Highlights
- 大多数 widget（checkbox、toggle、number_input、text_input/area、time/date_input、selectbox/multiselect、button 等）改用 `key` 作为主要标识
- 新增 `MultiselectColumn` 列类型
- `st.tabs` 支持 `default` 参数

## Notable Changes
- `st.write` 移除 `**kwargs` 支持
- `st.plotly_chart` 弃用 kwargs，改用 `config` 字典
- 主题新增主色、背景色、文字色配置

## 对当前项目的潜在影响 / 应用建议
- widget 以 key 为标识后，动态修改选项/默认值更可靠，对答题系统筛选器有益
- `st.tabs(default=...)` 可直接定位到“错题本”或“排行榜”标签页
