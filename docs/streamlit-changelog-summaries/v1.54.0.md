# Streamlit 1.54.0 更新摘要

**发布日期:** 2026-02-04

## Highlights
- 提供 `key` 时，可动态修改 `st.date_input`、`st.datetime_input`、`st.radio`、`st.select_slider`、`st.pydeck_chart` 等参数
- `st.dataframe` / Vega charts 的选择以 `key` 为主要标识
- `st.logo` 支持 Material 图标与 emoji

## Notable Changes
- 移除 experimental query params 与 `st.experimental_user`
- `add_rows` 显示浏览器弃用警告
- 为绑定 widget 到 query params 做准备

## 对当前项目的潜在影响 / 应用建议
- 动态修改选项让答题筛选器（如按章节动态更新题目）更易实现
- 移除 experimental query params 后，应迁移到稳定的 `st.query_params`
