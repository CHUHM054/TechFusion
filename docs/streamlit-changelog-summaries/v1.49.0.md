# Streamlit 1.49.0 更新摘要

**发布日期:** 2025-08-26

## Highlights
- `st.toast` 支持 `duration` 参数
- 新增 `st.pdf` 组件
- `st.file_uploader` 与 `st.chat_input` 支持目录上传

## Notable Changes
- 清理 experimental dialog / fragment / widget replay
- 弃用原生 `st.bokeh_chart`，推荐使用 `streamlit-bokeh` 组件
- `st.form_submit_button` 支持 `key`

## 对当前项目的潜在影响 / 应用建议
- `st.toast(duration=...)` 可控制答题提示停留时长
- `st.pdf` 可用于展示实验讲义或参考答案 PDF
