# Streamlit 1.31.0 更新摘要

**发布日期:** 2024-02-01

## Highlights
- 新增 `st.page_link`，可在多页应用中以编程方式生成页面链接
- 新增 `st.write_stream`，支持流式输出生成器或 OpenAI 类流式响应
- `st.chat_input` 支持行内（inline）使用

## Notable Changes
- 修复 Pandas 2.2.0 的 period 类型支持
- 支持 importlib-metadata v7
- Snowpark 依赖变为真正可选

## 对当前项目的潜在影响 / 应用建议
- `st.page_link` 可用于在答题系统各页面间建立导航，替代手写链接
- 若未来接入 AI 解析或流式提示，`st.write_stream` 可直接复用
