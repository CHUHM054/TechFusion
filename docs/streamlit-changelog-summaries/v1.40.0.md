# Streamlit 1.40.0 更新摘要

**发布日期:** 2024-11-06

## Highlights
- 正式发布 `st.pills` 与 `st.segmented_control` 按钮组组件
- `st.audio_input` 去实验化
- `st.image` 新增 `use_container_width` 参数

## Notable Changes
- 按钮组 `selection_mode` 的 `multiple` 重命名为 `multi`
- `st.text` 字体与溢出样式调整
- 支持在 Markdown 中插入 Streamlit Logo

## 对当前项目的潜在影响 / 应用建议
- `st.segmented_control` / `st.pills` 可替代部分 `st.radio` / `st.selectbox`，用于题型选择、难度筛选
- 按钮组视觉更紧凑，适合答题系统的顶部筛选栏
