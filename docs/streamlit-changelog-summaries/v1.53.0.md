# Streamlit 1.53.0 更新摘要

**发布日期:** 2026-01-14

## Highlights
- `st.metric` 支持 value/delta 参数使用 Markdown
- 侧边栏默认宽度可配置
- `st.chat_input` 文件上传 UI 改进（文件 chip、重试）

## Notable Changes
- `st.metric` 支持 delta 颜色调色板与 `format` 配置
- `st.select_slider` 选项支持 Markdown
- `st.data_editor` 可限制仅添加或仅删除行

## 对当前项目的潜在影响 / 应用建议
- `st.metric` Markdown 支持可在排行榜中显示带样式的正确率变化
- 侧边栏宽度配置可优化题库导航体验
