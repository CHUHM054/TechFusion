# Streamlit 1.48.0 更新摘要

**发布日期:** 2025-08-05

## Highlights
- `st.expander`、`st.status`、`st.spinner`、按钮、滑块采用全新设计
- `st.container` 支持 `direction`、`gap`、`horizontal_align`、`vertical_align` 等高级布局参数
- 支持不可关闭的 `st.dialog`（`dismissible=False`）

## Notable Changes
- `st.container` / `st.columns` 新增 width/height 参数（Breaking）
- 关闭 dialog 时可触发 rerun
- 滑块高度标准化

## 对当前项目的潜在影响 / 应用建议
- `st.status` 新设计可直接用于题库加载/保存状态展示
- `st.container` 布局参数可优化答题页面卡片排列
