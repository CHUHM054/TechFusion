# Streamlit 1.46.0 更新摘要

**发布日期:** 2025-06-18

## Highlights
- 大量组件新增 `width` 参数：`st.code`、`st.json`、`st.help`、`st.progress`、`st.slider`、`st.file_uploader`、`st.text_input`、`st.number_input`、`st.date_input` 等
- 允许嵌套 `st.expander` / `st.columns` / `st.popover` / `st.chat_message`

## Notable Changes
- 新增 `server.FolderWatchList` 配置
- `st.file_uploader` 等组件支持 `width` 后布局更灵活
- 修复 fragment 回调中修改元素的警告

## 对当前项目的潜在影响 / 应用建议
- 宽度参数让排行榜/错题本的表单和表格可更精细排版
- 嵌套容器可构建更复杂的题目解析卡片
