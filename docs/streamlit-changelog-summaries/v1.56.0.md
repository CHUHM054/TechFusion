# Streamlit 1.56.0 更新摘要

**发布日期:** 2026-03-31

## Highlights
- 新增 `st.menu_button` 组件
- `st.navigation` 支持 `expanded` 控制可见项数
- `st.Page` 支持外部 URL

## Notable Changes
- `st.dataframe` 支持程序化设置选择
- `st.file_uploader` / `st.chat_input` 支持文件类型快捷写法（如 'images'）
- `st.link_button` 支持 `on_click` rerun

## 对当前项目的潜在影响 / 应用建议
- `st.menu_button` 可用于答题系统顶部快捷操作菜单
- `st.Page` 外部 URL 可链接到实验教学资源页面
