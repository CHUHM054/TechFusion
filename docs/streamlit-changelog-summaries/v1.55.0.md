# Streamlit 1.55.0 更新摘要

**发布日期:** 2026-03-03

## Highlights
- widget 开始绑定到 URL query params（checkbox、toggle、text_input、color_picker 等）
- `st.pills` / `st.segmented_control` 提供 key 时支持动态修改选项
- `st.markdown` 新增 `width='auto'`

## Notable Changes
- `SnowparkConnection` 显示弃用警告
- 新增 `client.allowedOrigins` 配置
- `st.metric` 新增 `metricValueFontSize/Weight` 主题选项

## 对当前项目的潜在影响 / 应用建议
- query params 绑定可让答题筛选条件（科目/难度）直接反映在 URL，便于分享
- `st.pills` 动态选项适合根据已选科目动态切换章节筛选
