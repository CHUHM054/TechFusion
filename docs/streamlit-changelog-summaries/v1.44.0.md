# Streamlit 1.44.0 更新摘要

**发布日期:** 2025-03-25

## Highlights
- 新增 `st.badge` 与 Markdown badge 语法
- 新增 `streamlit init` CLI 命令
- React 前端升级至 React 18 createRoot

## Notable Changes
- 主题新增 `headingFont`、`sidebar`、`codeBackgroundColor` 等选项
- `st.table` 支持 Pandas styler tooltips
- 默认切换到 MPAv2

## 对当前项目的潜在影响 / 应用建议
- `st.badge` 可用于展示答题状态标签（如“已通过”“待复习”）
- 主题细化后可进一步定制答题系统深色/浅色模式
