# Streamlit 1.51.0 更新摘要

**发布日期:** 2025-10-29

## Highlights
- `st.plotly_chart`、`st.vega_lite_chart`、`st.pydeck_chart`、`st.altair_chart` 新增 width/height 参数
- 行选择激活时 `st.dataframe` 自动隐藏行索引
- `st.popover` 新增 `type` 参数匹配按钮样式

## Notable Changes
- 新增 `codeTextColor` 主题配置
- `theme.base` 支持可复用自定义主题
- `streamlit run` 无参数时默认运行 `streamlit_app.py`

## 对当前项目的潜在影响 / 应用建议
- 图表 width/height 参数便于在智能分析页控制雷达图/柱状图尺寸
- 可复用主题便于在多个物理实验科目间统一视觉
