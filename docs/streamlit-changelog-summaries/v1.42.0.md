# Streamlit 1.42.0 更新摘要

**发布日期:** 2025-02-04

## Highlights
- 新增原生 `Auth` 认证功能
- `st.spinner` 显示已运行时间
- `st.dataframe` / `st.data_editor` 支持从 UI 排序、固定、隐藏列

## Notable Changes
- 图表默认启用 `use_container_width=True`
- `st.table` / `st.image` caption 支持 Markdown
- 移除从 `X-Streamlit-User` header 读取用户信息

## 对当前项目的潜在影响 / 应用建议
- 排行榜/错题本的 dataframe 可直接让用户交互排序，减少代码
- `st.spinner` 时间显示便于用户感知加载耗时
